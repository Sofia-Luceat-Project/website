## はじめに

このページは、「ウェブサイトの記事を書いてみたいけど、やり方が分からない…」という方向けのガイドです。
専門的な知識は一切不要です。パソコンの基本的な操作ができれば、誰でも簡単に記事を作成し、公開できます。

一緒に手順を追いながら、あなたの最初の記事を完成させましょう！

---

## ステップ1：記事ファイルの準備

まず、記事の「原稿用紙」となるファイルを用意します。

1.  **フォルダを開く**
    PCのエクスプローラーで、以下のフォルダを開きます。
    `website\src\content\posts`

2.  **記事の種類を選ぶ**
    `posts`フォルダの中には、`Blog`や`Wiki`といった、記事の種類ごとの小部屋（サブフォルダ）があります。書きたい記事の種類に合った部屋を選んでください。

3.  **新しいファイルを作成する**
    選んだフォルダの中で、右クリックから「新規作成」→「テキスト ドキュメント」を選び、新しいファイルを作成します。

4.  **ファイル名を付ける**
    ファイル名は、**半角の英数字とハイフン (`-`)** を使って、記事の内容が分かりやすい名前を付けます。最後に、ファイルの拡張子を `.txt` から `.md` に変更してください。

    - **良い例:** `my-first-blog.md`, `dark-survivor-guide-1.md`
    - **悪い例:** `最初のブログ.md` (日本語は使えません), `my first blog.md` (スペースは使えません)

これで、記事を書くための「原稿用紙」が準備できました！

---

## ステップ2：記事の基本情報を入力する (Frontmatter)

次に、作成したファイルをZedなどのテキストエディタで開き、記事の「プロフィール」を設定します。
これは「Frontmatter（フロントマター）」と呼ばれるもので、記事のタイトルや公開日などをサイトに教えるための大切な情報です。

ファイルの**一番最初**に、以下のテキストをコピーして貼り付けてください。

```yaml
---
title: ここに記事のタイトルを入力
published: 2025-11-15
description: "ここに記事の簡単な説明文を入力（100文字程度がおすすめです）"
image: ./index.jpeg
tags: [キーワード1, キーワード2]
category: Blog
draft: false
series: Example
---
```


そして、各項目の `"` の中を、あなたの記事に合わせて書き換えます。

- `title`: 記事のタイトルです。読者が一番最初に目にする、最も重要な部分です。
- `published`: 記事を公開する日付です。`YYYY-MM-DD` の形式で入力します。
- `description`: 記事の要約です。検索結果などにも表示されることがあります。
- `image`: 記事のサムネイル画像です。`./` から始まる相対パスで指定します。
- `tags`: 記事に関連するキーワードです。複数ある場合は `,` で区切ります。
- `category`: 記事のカテゴリーです。`Blog` や `Wiki` など、ステップ1で選んだフォルダ名と同じにすると分かりやすいです。
- `draft`: `false` にしておくと公開記事になります。もし下書きにしたい場合は `true` にします。
- `series`: 記事のシリーズ名です。

## ステップ3：本文を書く (Markdown記法)

基本情報の下に、いよいよ記事の本文を書いていきます。
ウェブサイトの記事では、「Markdown（マークダウン）」という簡単な記法を使って、文章に見た目を付けていきます。

### 見出し

見出しは、文章の「章」や「節」を作るためのものです。`#`（シャープ）の数で大きさが変わります。

**書き方:**
```markdown
# 一番大きな見出し
## 中くらいの見出し
### 小さな見出し
```

**表示例:**
# 一番大きな見出し
## 中くらいの見出し
### 小さな見出し

### 文字の強調（太字）

文章の中で特に伝えたい部分は、`**`（アスタリスク2つ）で囲むと**太字**になります。

**書き方:**
```markdown
これは **とても重要** なポイントです。
```

**表示例:**
これは **とても重要** なポイントです。

### 箇条書き

情報を整理して伝えたいときは、箇条書きが便利です。文の先頭に `-`（ハイフン）を付けます。

**書き方:**
```markdown
- りんご
- ごりら
- らっぱ
```

**表示例:**
- りんご
- ごりら
- らっぱ

### リンク

他のウェブサイトを紹介したいときは、リンクを使いましょう。

**書き方:**
```markdown
[Google](https://www.google.com/)
```

**表示例:**
[Google](https://www.google.com/)

### 便利な装飾機能

このサイトでは、さらに見た目をリッチにするための特別な装飾が使えます。

#### 囲み枠（アドモニション）

メモやヒント、注意点などを目立たせるための囲み枠です。

**書き方:**
```markdown
:::note
ここにメモしたいことを書きます。
:::

:::tip[ヒント]
豆知識などを書くのに便利です。タイトルも付けられます。
:::

:::important
読者に必ず読んでほしい重要な情報を書きます。
:::
```

**表示例:**
:::note
ここにメモしたいことを書きます。
:::

:::tip[ヒント]
豆知識などを書くのに便利です。タイトルも付けられます。
:::

:::important
読者に必ず読んでほしい重要な情報を書きます。
:::

#### リンクカード

ウェブサイトへのリンクを、かっこいいカード形式で表示できます。

**書き方:**
```markdown
::link-card{url="https"//www.redotengine.org/" title="Redot Engine" description="Game Engine, Game Changer"}
```

**表示例:**
::link-card{url="https://www.redotengine.org/" title="Redot Engine" description="Game Engine, Game Changer"}

## やってみよう！記事作成の例

これまでの内容をまとめた、記事のサンプルです。
これをコピーして、自分の記事を作る際の参考にしてみてください。

```markdown
---
title: "私の好きなゲームについて"
published: "2025-11-15"
description: "私が最近ハマっているゲームの魅力を紹介します！"
tags: ["ゲーム", "レビュー"]
category: "Blog"
draft: false
---

## はじめに

こんにちは！今日は私が夢中になっているゲーム『Dark Survivor』について語りたいと思います。

## 『Dark Survivor』の魅力

このゲームの魅力は、なんといってもその**爽快感**にあります！

- 大量の敵をなぎ倒す楽しさ
- レベルアップでどんどん強くなる成長要素
- スキルの組み合わせを考える戦略性

:::tip[ここがポイント！]
最初は難しく感じるかもしれませんが、何度かプレイするうちにコツが掴めてきて、どんどん楽しくなります！
:::
```
