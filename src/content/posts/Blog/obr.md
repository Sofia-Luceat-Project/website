# 🌐 OBR  
**オフライン対応・サイト保存ブラウザ**

OBR は、  
「あとで読む」を本気で快適にするために作られた  
**サイト保存対応ブラウザ**です。

一度開いたページを保存しておけば、  
ネットがない環境でも、いつもの見た目のまま閲覧できます。

---

## ✨ OBR でできること

### 🔗 普通のブラウザとして使える
- URL検索・ページ移動
- 戻る / 進む / 再読み込み
- JavaScript 対応ページもそのまま表示

### 💾 閲覧しながらサイトを保存
- 見ているページを自動で保存
- HTML / CSS / JS / 画像 / フォントをまとめて記録
- 保存操作は邪魔にならない裏側処理

### 📦 オフラインでそのまま読める
- ネット未接続でも表示可能
- レイアウト崩れを最小限に抑制
- ローカルデータのみを使用

---

## 🔄 オンライン / オフラインを完全に分離

OBR には **2つの明確なモード**があります。

### 🌍 オンラインモード
- 常にインターネットから取得
- 保存済みデータは参照しない
- 普通のブラウザと同じ操作感

### 📂 オフラインモード
- ネットワークを使用しない
- 保存済みサイトのみを表示
- 「読む」ことに特化した表示

> 今見ているページが  
> 「ネット由来なのか」「保存データなのか」  
> が曖昧になることはありません。

---

## 🗂 サイト単位で管理

保存データは **サイトごと**に整理されます。

- ドメイン単位で一覧表示
- 保存日時・ページ数を確認
- ワンクリックで再表示

技術ドキュメント、攻略サイト、個人ブログなど  
「あとで読み返したい情報」の整理に最適です。

---

## 🚫 広告・トラッカーを保存しない

OBR は、保存時に以下を行います。

- 広告リクエストをブロック
- トラッカー通信を遮断
- 不要な iframe を保存しない

これにより：

- 保存サイズ削減
- オフライン表示の安定化
- 読書体験の向上

を実現しています。

---

## 🧠 こんな人におすすめ

- 技術ドキュメントをローカルに残したい
- ネット環境が不安定な場所で作業する
- 後からじっくり読み返したい記事が多い
- Wayback やクローラ系ツールが重すぎると感じる

---

## 🛠 技術スタック（概要）

- **Electrobun**
- ネイティブ BrowserView（iframe 不使用）
- ローカルアーカイブ管理
- Windows 対応（予定）

## ▶ まずはページを開いてみてください

URL を入力して「移動」を押すだけ。  
OBR は、あなたが読んだページを静かに記憶します。

**「読む」ためのアーカイブブラウザを、あなたの手元に。**
